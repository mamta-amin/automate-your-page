World Wide Web

World Wide Web is collection of web pages which are made up of text files. These files contain image, text, links to other pages, video, and HTML tags. A browser uses these text documents along with other files for styling and scripting to display what we see on the screen.

HTML Basics

HTML (Hypertext Markup Language) is a language used by a browser to display a web page. HTML is made up of series of tags that hold instructions for the computer. Tags are basically commands for the browser. HTML document can only contain text and can be created using editors like Sublime, Notepad++, etc. As you write any code it is important to remember not to repeat code since all that code has to be maintained. Don't Repeat Yourself (DRY) is one of the fundamental principal of programming. It is also important that when writing HTML code you should use logical approach in structuring your page. You can do this by grouping related content within an HTML element. For example, I have broken my page into four groups using <div> element as follows: header, nav, section, and footer.
Each web page has a basic structure that consists of some base tags like HTML, head, and body. Other sub tags branch out from these base tags forming a tree like structure. Most tags come in pair of an opening and closing tags; however there are some that have no closing tags. Below are some examples of tags and their function. Cascading Style Sheet

CSS (Cascading Style Sheet) is a language used to create presentation of a webpages. While HTML allows you to structure your content CSS allows you to keep your content separate from your styling. Style can be included in a web page by linking the CSS file in the HTML document or by writing style in-line with HTML. Last option is not preferred since it may cause a lot of repeated code. When you create CSS file, it is easier and cleaner to have same class name for similar elements. This is because when you create your CSS file you only have to write your styles for that one class instead of writing same style multiple times.In my webpage, I have created a class called "notes" that is used several times in my HTML page and have applied same formatting. Python

Python is a programming language. When code writtein in Python is executed, Python interpreter converts the code into set of instructions. Since computers are dumb, code that you write has to be granatically correct in Python language.

Functions

Functions are block of code that is self contained. It takes some input, does processing based on instructions provided in the code, and return some output. Function block starts wiht "def" key word followed by function name and "()" which can contain variables.

Strings

Strings are created when you enclose characters in quotes. Strings can be manipulated in Python using various built in functions like: find, len, [] with index number for getting substring

Structures

Data structures basiclly hold related data. Python has different types of data structures that include lists, tuples, dictionaries, etc.
Lists contain orderd items and can contain values of any type. Lists are also mutable which means they can be modified. Some list operators are 'append', '+'' and 'len'.
* Append- is a method that adds an element at the end of a list. Syntax for append is '<list>>.append(<element>').
* '+' takes two lists and produces new list combining elements from both lists. It does not mutate the list.
* 'len' is another list operator that gives number of elements in a list.
* 'index' - finds the position of a value in a list

Loops

Loops - are usefull when you want to execute same code until certain condition is met. Using Loop reduces repeated code. Loops can be exited by using 'break' statement. Break statement resumes code at next statement in your block. Other statements like 'pass' and 'continue' are also used to exit a loop. The two main types of loop are as follows:
* while loop - run the code while a condition is true.
* for loop - runs the code multiple times.
